## <a id="omnivore"></a>옴니보어 (Omnivore) 🦎

**"나이, 성별, 소득 경계를 허무는 잡식성 소비자의 등장"**

'잡식 동물'을 뜻하는 옴니보어는 고정관념에 얽매이지 않고 자신만의 취향에 따라 자유롭게 소비하는 현대 소비자를 의미합니다.

### 주요 특징
- **세대 경계 파괴**: 60대가 틱톡 1위 이용 앱, MZ세대가 트로트 열광
- **성별 구분 무의미**: 남성 화장품, 여성 위스키 시장 급성장
- **다양한 관심사**: 한 사람이 명품과 다이소를 동시에 소비

> 💡 **마케팅 인사이트**: 기존의 인구통계학적 타겟팅보다 '취향 기반 타겟팅'이 중요해졌습니다.

## <a id="aboha"></a>아보하 (Aboha) 🌿

**"아무것도 하지 않고 보통의 하루를 살고 싶다"**

특별한 일이 없어도 평범한 하루에 만족하며, 작은 행복을 소중하게 여기는 삶의 태도입니다.

### 왜 뜨고 있을까?
- **번아웃 시대**: 과도한 경쟁과 성과주의에 지친 현대인
- **작은 행복**: 카페에서 책 읽기, 산책, 홈카페 즐기기
- **느린 삶 추구**: '갓생'(God-생)에서 '아보하'로 전환

### 관련 트렌드
| 키워드 | 의미 |
|--------|------|
| 소확행 | 소소하지만 확실한 행복 |
| 휘게 | 덴마크식 여유로운 삶 |
| 워라밸 | 일과 삶의 균형 |

## <a id="topping"></a>토핑경제 (Topping Economy) 🍕

**"본품보다 토핑에 더 돈을 쓴다!"**

마라탕처럼 원하는 토핑을 더하는 소비 방식이 모든 산업으로 확산되고 있습니다.

### 토핑경제 예시
- **음식**: 마라탕, 요거트, 버블티 토핑 추가
- **패션**: 기본템 + 커스텀 악세서리
- **자동차**: 기본 차량 + 개인 맞춤 옵션
- **스마트폰**: 케이스, 그립톡, 스티커 데코

### 성공 포인트
```
기본값 (저렴하게) + 토핑 (프리미엄) = 나만의 특별함
```

## <a id="harmless"></a>무해력 (Power of Harmlessness) 🐰

**"자극적인 것보다 부드럽고 순한 것이 좋아"**

공격적이지 않은 부드러운 영향력, '무해한' 콘텐츠와 서비스가 주목받고 있습니다.

### 무해력 트렌드
- **캐릭터**: 잔망루피, 포챠코, 망그러진 곰 등 순한 캐릭터 인기
- **콘텐츠**: 힐링 예능, ASMR, 브이로그
- **마케팅**: 공격적 광고 → 감성적 스토리텔링
- **SNS**: 악플 대신 응원 문화

> 🌸 과도한 자극에 지친 현대인들이 '무해함'에서 안식을 찾고 있습니다.

## <a id="practical"></a>실용소비 & 안티플렉스 💰

**"플렉스는 그만! 실속 있는 소비가 진짜 힙"**

고물가 시대, 과시보다 실용을 중시하는 소비 패턴이 확산되고 있습니다.

### 실용소비 데이터
| 항목 | 변화 |
|------|------|
| 체크카드 사용 | 2030세대 급증 |
| 중고거래 앱 | 이용자 크게 증가 |
| 중고차 구매 | 전년 대비 29% 상승 |
| 불필요한 구매 자제 | 80.7% 응답 |

### 실용소비 트렌드
- **요노(YONO)**: You Only Need One - 필요한 것만!
- **듀프(Dupe)**: 명품 대신 비슷한 저렴이
- **리셀(Resell)**: 중고 거래 활성화
- **구독경제**: 소유보다 경험 중시

## <a id="ai"></a>AI 시대 & 제로클릭 🤖

**"AI가 모든 것을 바꾸고 있다"**

2025년 가장 뜨거운 키워드! AI가 일상과 비즈니스 전반을 혁신하고 있습니다.

### AI 트렌드 키워드
- **제로클릭**: AI가 바로 답을 제공, 클릭 필요 없음
- **AX(AI Transformation)**: AI 기반 조직 구조 변화
- **페이스테크**: 얼굴 인식 기술로 개인화 서비스
- **AI 에이전트**: 자율적으로 업무 수행하는 AI

### AI 시대 준비하기
```
✅ AI 도구 활용 능력 키우기
✅ 창의성과 감성 역량 강화
✅ AI와 협업하는 방법 배우기
✅ 인간만의 강점 발휘하기
```
